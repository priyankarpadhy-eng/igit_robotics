# IGIT Robotics News Portal

A premium, high-fidelity news and innovation portal for the Indira Gandhi Institute of Technology, Sarang. Built with a focus on editorial excellence and technical robustness.

## Tech Stack
- **Frontend**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + Custom Vanilla CSS (Newspaper Aesthetic)
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Animations**: CSS Transitions + GSAP ready

## Design System
The portal utilizes a "Paperio" inspired design language:
- **Colors**: Beige Paper (#F4F1EA), Charcoal Carbon (#1C1C1C), Alert Red (#BC2C2C).
- **Typography**: 
  - `Instrument Serif`: For the main masthead and hero accents.
  - `Playfair Display`: For editorial headlines.
  - `Inter`: For high-readability body text.
- **Texture**: Subtle SVG noise grain for a tactile paper feel.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Backend configuration is located in `src/lib/firebase-config.ts`.
